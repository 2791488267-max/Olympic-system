Olympic Medal Management System 

Based on the Olympic medal information management and ranking system developed by C++, it supports national management, medal entry and real-time ranking query.

🚀 Quick download

Go to [Releases] (https://github.com/2791488267-max/Olympic-system/releases/tag/v1.0.0) page to download the latest version:

- Application: `Olympic-system.zip` (double-click exe to run after decompression)

- Source code: `Source code (zip)` or `Source code (tar.gz)`

✨ Core functions

1. National Information Management: Add and edit the information of participating countries

2. Medal data operation: Enter the number of gold, silver and bronze medals in each country

3. Real-time ranking query: Automatically generate a ranking list according to the number of medals

🖥️ Operating instructions

1. Download `Olympic-system.zip` and unzip it to any folder

2. Double-click the executable file launcher after decompression

3. System language: Simplified Chinese

4. Operating environment: Windows system (no additional installation dependency required)

🛠️ Development information

- Development language: C++

- Project Type: Desktop Application
